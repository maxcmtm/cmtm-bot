# הודעה לתמיכת/מפתחי Wizup (להעתקה)

שלום, אני מחבר בוט AI חיצוני שיענה אוטומטית ללידים בוואטסאפ דרך המספר שלי אצלכם
(חבילת Basic Plus). הבוט הוא שרת חיצוני שמקבל הודעה, מייצר תשובה, ומחזיר אותה.
אשמח לעזרה ב-2 דברים:

## מסלול א' — דרך Flow Builder (מעדיף להתחיל בזה)
אני רוצה לבנות פלואו: טריגר "הודעה נכנסת" → נוד HTTP Request
(POST ל-URL שלי, עם header וגוף JSON) → לקחת שדה מתוך תשובת ה-JSON
ולשלוח אותו חזרה ב-"שלח הודעה".

1. האם נוד ה-HTTP Request תומך בקריאת JSON response ובמיפוי שדה ממנו
   (למשל `reply`) למשתנה שאפשר לשלוח בהודעה? איך עושים זאת בדיוק?
2. מה שמות המשתנים בפלואו ל: טקסט ההודעה הנכנסת, טלפון השולח, שם השולח?

## מסלול ב' — חיבור ישיר דרך API (לטווח הארוך)
1. האם יש webhook שאפשר להגדיר שישלח כל הודעה נכנסת לכתובת URL שלי? איך מגדירים?
2. האם יש endpoint ב-API לשליחת הודעת session (טקסט חופשי, בתוך חלון 24 שעות)
   לפי מספר טלפון? מה ה-endpoint, ואיך מקבלים טוקן מפתחים?
3. יש תיעוד API / מפתחים שתוכלו לשלוח לי?

תודה!

## (לשימוש פנימי — לא לשלוח) פרטי החיבור שכבר מוכנים אצלנו
- ה-webhook של הבוט מקבל POST עם:
  - Header: `x-bot-secret: cmtm-wizup-7421`
  - Body: `{ "message": "<טקסט>", "lead": { "id": "<טלפון>", "name": "<שם>" } }`
  - מחזיר JSON: `{ "reply": "...", "persona": "...", "new_score": N, "handoff": true/false }`
- כל מה שצריך מ-Wizup: לשלוח לנו את ההודעה הנכנסת, ולשלוח את `reply` חזרה ללקוח.
