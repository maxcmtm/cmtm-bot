# מדריך: לחבר את הבוט ל-Wizup (Flow Builder) — צעד-צעד

המטרה: כשליד כותב למספר שלך ב-Wizup → Wizup קורא לבוט שלנו → נועה עונה.

---

## לפני שמתחילים — לוודא שהבוט חי
בטרמינל:
```bash
bash ~/cmtm-leads-bot/start-bot.sh
```
זה ידפיס כתובת כמו `https://xxxx.trycloudflare.com/webhook`.
**העתק את הכתובת הזו** — תצטרך אותה בשלב 3.
(הכתובת משתנה בכל הפעלה מחדש. אם הפעלת מחדש — עדכן אותה ב-Wizup.)

---

## בניית הפלואו ב-Wizup

### שלב 1 — צור פלואו חדש
- ב-Wizup → תפריט **בוטים / Flow Builder / אוטומציות** → **צור בוט/פלואו חדש**.

### שלב 2 — טריגר: הודעה נכנסת
- בחר טריגר מסוג **"הודעה נכנסת"** (Incoming message) — שיופעל על **כל הודעה** (בלי מילת מפתח).

### שלב 3 — נוד "HTTP Request" (הקריאה לבוט)
הוסף נוד **HTTP Request** והגדר:
- **Method:** `POST`
- **URL:** הכתובת מהסקריפט, למשל `https://xxxx.trycloudflare.com/webhook`
- **Headers:**
  - `content-type` = `application/json`
  - `x-bot-secret` = `cmtm-wizup-7421`
- **Body** (JSON) — השתמש במשתנים של Wizup לטקסט ההודעה ולשם/טלפון השולח:
  ```json
  {
    "message": "{{message.text}}",
    "lead": { "id": "{{contact.phone}}", "name": "{{contact.name}}" }
  }
  ```
  > שמות המשתנים המדויקים ({{message.text}} וכו') עשויים להיות מעט שונים ב-Wizup —
  > בחר מרשימת המשתנים שלהם את "טקסט ההודעה", "טלפון", "שם".

### שלב 4 — שמור את התשובה
- אחרי קריאת ה-HTTP, התשובה היא JSON שמכיל את השדה **`reply`**.
- ב-Wizup, מפו/שמור את `reply` (לפעמים נקרא "response.reply" או דרך "Map response") למשתנה — נקרא לו `bot_reply`.

### שלב 5 — נוד "שלח הודעה"
- הוסף נוד **Send Message** ושלח את `{{bot_reply}}`.

### שלב 6 — (אופציונלי, מומלץ) העברה לנציג
- הוסף **Condition**: אם `handoff` מהתשובה = `true` →
  - תייג את השיחה `נציג` / הקצה ליועצת / שלח התראה לצוות.

### שלב 7 — הפעל (Activate)
- שמור והפעל את הפלואו.

## בדיקה
1. מהטלפון הפרטי שלך → שלח הודעה למספר ה-Wizup, למשל: *"היי, כמה עולה הקורס?"*
2. אמור לחזור: נועה לא נוקבת מחיר ומציעה שיחת התאמה.
3. נסה עוד: *"יש לי ילד עם חרדות"*, *"זה לא שטויות?"*, *"אני רוצה נציג"*.

## אם נתקע
- שלח לי צילום מסך של המסך שבו נתקעת ואני אכוון בדיוק.
- בדוק שהבוט חי: בדפדפן פתח את `https://xxxx.trycloudflare.com/health` — אמור להחזיר `{"ok":true...}`.

## חשוב לזכור (שלב בדיקות)
- הבוט רץ על המחשב שלך — **המחשב צריך להישאר פתוח** והסקריפט לרוץ.
- הכתובת זמנית ומשתנה בהפעלה מחדש.
- ➜ ברגע שהבדיקות טובות, נארח את הבוט ב-Render (כתובת קבועה, רץ 24/7 בלי המחשב שלך).
